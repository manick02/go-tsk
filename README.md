# go-tsk

A simple Go project with proper testing setup.

## Project Structure

```
go-tsk/
├── cmd/
│   └── app/
│       └── main.go
├── pkg/
│   └── calculator/
│       ├── calculator.go
│       └── calculator_test.go
└── internal/
```

## Running Tests

To run tests:

```bash
go test ./...
```

## Running the Application

To run the application:

```bash
go run cmd/app/main.go
```
